# dump

以十六进制格式转储指定内存区域。

## 用法

```shell
xrock dump <address> <length>
```

## 参数

| 参数 | 说明 |
|------|------|
| `<address>` | 内存起始地址 |
| `<length>` | 读取的字节数 |

## 说明

读取设备内存并以带地址、十六进制值与 ASCII 字符的格式输出。

:::note
内存地址从 0 开始，**不是** SDRAM 的物理地址。
:::

## 示例

转储起始地址 `0x1000` 处的 256 字节：

```shell
xrock dump 0x1000 256
```
