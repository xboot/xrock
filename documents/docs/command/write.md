# write

将本地文件写入设备内存，写入过程带进度显示。

## 用法

```shell
xrock write <address> <file>
```

## 参数

| 参数 | 说明 |
|------|------|
| `<address>` | 内存起始地址 |
| `<file>` | 待写入的本地文件路径 |

## 说明

读取本地文件完整内容并写入设备内存的指定地址，传输过程显示进度条。常用于将固件加载到内存后配合 `exec` 执行。

:::note
内存地址从 0 开始，**不是** SDRAM 的物理地址。
:::

## 示例

将 `xstar.bin` 写入内存地址 `0x60000000`：

```shell
xrock write 0x60000000 xstar.bin
```
